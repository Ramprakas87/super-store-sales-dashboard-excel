## super-store-sales-dashboard-excel
This project presents an interactive Superstore Sales Dashboard built in Microsoft Excel to analyze sales performance, profit trends, and customer insights. The dashboard transforms raw transactional data into meaningful business insights using data cleaning, pivot tables, and dynamic visualizations.

#  Super Store Sales Dashboard (Excel)

An interactive sales dashboard created using Microsoft Excel to analyze sales performance, profit trends, and regional insights.

---

##  Project Overview  

This Excel dashboard provides insights into:

-  Total Sales & Profit Analysis  
-  Region-wise Performance  
-  Category & Sub-Category Analysis  
-  Monthly Sales Trends  
-  Customer Segment Insights
